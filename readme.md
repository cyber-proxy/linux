## 开发环境搭建
* 准备一个centos云主机
* 在windows下安装putty和visual Code
* centos安装git
* centos安装g++
```cmd
    yum install gcc gcc-c++ kernel-devel
    yum groupinstall "Development Tools"
    yum install make
```
* centos安装fpt服务
```text
    1、
    2、
    3、
```
* 将centos共享的fpt目录映射到windows下
